# 🎲 Python Matka Game

A simple command-line Matka style number game written in Python.

The game randomly selects cards, generates numbers, and calculates the final winning digit based on the total.

---

## Run

### Mac / Linux

```bash
python3 main.py
```

### Windows

```bash
python main.py
```

---

## How It Works

- 3 random cards are selected
- Each card gets a random value
- All values are added together
- The final winning digit is calculated from the total
- If your bet matches the winning digit, you win
