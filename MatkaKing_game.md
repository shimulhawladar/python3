# 🎲 Matka King Game

A simple Python terminal based Matka King game built using clean and scalable Python code.

The game randomly picks 3 unique cards, generates random values, calculates the winning number, and checks whether the player's bet matches the result.

- Object-oriented programming (OOP)

# 🎮 Game Rules

1. The system randomly selects 3 unique cards.
2. Each card receives a random number between 1 and 10.
3. All values are added together.
4. The winning number is calculated:
   - If total = 10, 20, or 30 → first digit is used
   - Otherwise → last digit is used
5. If your bet matches the winning number, you win.
